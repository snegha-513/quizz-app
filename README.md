# Quiz App Using Tkinter

## Description
A GUI-based quiz application developed using Python Tkinter.
It allows users to answer MCQs and view their final score.

## Features
- MCQs with 4 options
- Next and Previous navigation
- Score calculation
- Simple UI

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python
2. Run quiz app.py
3. Answer questions and view score
